Pour fonctionner avec JSPM:

Vérifier si la version la plus récente de typescript contient la fonctionalité:
npm update -g typescript
tsc --help | grep baseUrl

Sinon:
Installer la future version de typescript:
npm install -g typescript@next

La version courante (0.10.6) de VSCode ne gère pas cette fonctionnalité.

Mettre à jour le path de tsc server pour les différentes ide//plugins:
https://github.com/Microsoft/TypeScript/wiki/Nightly-drops
