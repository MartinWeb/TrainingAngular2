Ecosystem ES6 Introduction - Rest Operator instead of arguments keyword

arguments allow to play with parameters passed when a function, regardless the parameters really defined in the function

var getSum= function(unusedParam){ var sum= 0; for (int i= 0; i < arguments.length; i++){ sum+= arguments[i]; } };

The rest operator ( ... ) is more clear and let us work with a specified parameter name. It can also be used when destructuring data.

var getSum= function(...toAdd){ var sum= 0; for (int i= 0; i < toAdd.length; i++){ sum+= toAdd[i]; } }; var getSumButFirst= function(...toAdd){ let [first, ...next] = toAdd; for (int i= 0; i < next.length; i++){ sum+= next[i]; } };
