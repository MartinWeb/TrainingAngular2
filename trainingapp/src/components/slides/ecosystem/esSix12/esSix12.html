Ecosystem ES6 Introduction - Modules (...)

ES6 now provides its own modules implementation, keeping in mind the behavior of AMD and CommonJS. In other words, this implies:

  • An easy and understandable syntax
  • reccognizable by tools ( such as IDE to get auto completion for example)
  • Adapted for web browsers (so downloaded asynchronously)
  • Cyclic dependencies detection

Modules will be everywhere with ES6 and so with Angular2
