Ecosystem Typescript - Basics about types

typescript provides the basic types we are used to work with on most languages:

let firstName: string= 'John'; let age: number = 10; let isMarried: boolean = true; let people: Array<Person> = new Array<Person>(); // or Person[]

Enumerations are also available to create lists of values

enum Gender { MALE, FEMALE } let gender: Gender= Gender.MALE;

The any type can deal with ... any type ...

let sample: any = 'Hello world'; sample = 1; sample = Gender.FEMALE;

You can also work with unions to constrain to a few types only

let multipleTypes: boolean | string = true; multipleTypes = 'Hello world';
