Ecosystem ES6 Introduction - Destructuring Assignment

This is a new syntax which can be surprising when you read it for the first time, so it's interesting to have a look at it. The idea is to assign variables on a shortest way.

Ex√©cuter tout le code pour valider les exemples

//ES5 var person= {firstName: 'bob', lastName:'dylan'}; var pfirstName= person.firstName; var plastName= person.lastName; //ES6 let person= { firstName: 'bob', lastName:'dylan' }; let { firstName: pfirstName, lastName: plastName } = person; //Or even if the variable names as the field one let { firstName, lastName } = person; //And it works at any level let person= {id: 1, firstName: 'bob', lastName:'dylan', bestFriend: { id: 2}}; let {id, firstName, lastName, bestFirend:{id:bestFriendId}}= person; console.log(bestFriendId); //Finally it works as well with arrays let friendsIds= [1, 4, 5]; let [firstFriend, secondFriend]= friendsIds; //firstFriend= 1 //secondFriend= 4
