Preamble About Web Components - Shadow Dom

  • Template content is not inserted between the component tags in the rendered DOM
  • But it is encapsulated into the component, with the one of the Shadow Dom

In the example below, we display shadow content, but if we inspect the DOM content, let's see DOM content. The shadow content is so in the Shadow Dom

<template id="popupTemplate"> <div>Shadow DOM</div> </template> <mypopup>DOM content</mypopup> var shadow = document.querySelector('mypopup').createShadowRoot(); var template = document.querySelector('#popupTemplate'); var clone = document.importNode(template.content, true); shadow.appendChild(clone);

The Shadow Dom encapsulation allows to make the component independant of the DOM and so reusable. For example, the global document stylesheets will not apply to it, and you can't access its elements with document.getElementById
