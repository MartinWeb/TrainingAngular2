Preamble About Web Components - Composition

An important aspect of html is composition. For example, the <ul> tag can be composed with the <li> one.

<ul> <li></li> <li></li> </ul>

But to achieve this with Web Components, we need a new tag: <content>

<template id="popupTemplate"> <p> Here is the content: </p> <content></content> </template> <mypopup>Commposed Content</mypopup> var shadow = document.querySelector('mypopup').createShadowRoot(); var template = document.querySelector('#popupTemplate'); var clone = document.importNode(template.content, true); shadow.appendChild(clone);

Finally, you can target the content you want and use as much content tags as you want

<template id="popupTemplate"> <h1> <content select="#title" > </content> </h1> <p> <content select="#text" > </content> </p> </template> <mypopup> <span id="title">My Popup Title</span> <span id="text">My Popup Content</span> </mypopup> var shadow = document.querySelector('mypopup').createShadowRoot(); var template = document.querySelector('#popupTemplate'); var clone = document.importNode(template.content, true); shadow.appendChild(clone);
