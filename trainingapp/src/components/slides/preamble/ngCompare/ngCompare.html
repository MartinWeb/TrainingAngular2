Preamble Angular2 VS Angular1, what remains? How to port from 1 to 2?

  • Angular2 is really different from Angular1, no more:
    • scope
    • digest cycle
    • 2 way data binding
    • controllers
  • Some concepts remain, even if their implementation is fully different
    • Template expressions, {{expression}}, one way data binding
    • 2 way data binding can be simulated with help of events
    • services
    • The notions of directive, even if its name and usage change
    • Automatic change detection is still available, but is managed differently by the framework (events)
    • filters are now named pipes
  • To port from Angular1 to Angular2, nothing magic, 3 important things:
    1. Prepare the existing application! Which means follow best practices, make it use recent technologies such as typescript/ES6, and use the angular2 router ported to angular1
    2. Use angular1 directives, which will ease the process of converting angular1 directives to angular2 components
    3. Include Angular2 in your Angular1 application, to make them coesxist together and convert things incrementally
