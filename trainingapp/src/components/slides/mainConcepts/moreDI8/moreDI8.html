Main Concepts Dependency Injection - More Details - Component

And then, to take advantage of the concept of inheritance between providers, we have the providers attribute on the @Component annotation.

import {Component} from 'angular2/core'; import {bootstrap} from 'angular2/platform/browser'; import {Person} from 'services/Person'; @Component({ selector: 'main-cmp', template: 'Hello World: <sub-cmp></sub-cmp>' }) class MainCmp{ constructor(public person: Person){} } bootstrap(App, [Person]); import {Component} from 'angular2/core'; import {Person} from 'services/Person'; @Component({ selector: 'sub-cmp', template: `I am a sub component and I also have child components:<grandchild-cmp></grandchild-cmp> `, providers: [Person] }) class SubCmp{ constructor(public person: Person){} }

At this point, SubCmp and all its children components will use a new instance of Person, not the one defined at the bootstrap level.
