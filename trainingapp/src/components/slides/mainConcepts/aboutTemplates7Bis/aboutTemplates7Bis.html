Main Concepts About Templates - Property Binding - Declarations

  • In previous examples, we used the @Input annotation to declare a parameter for a component
  • And you can use the inputs attribute in the @Component annotation as well
import {Component, Input} from 'angular2/core'; @Component({ selector: 'cmp', template: ` <h1> Hello {{name}}</h1> ` }) export class Cmp { @Input() name: string; } import {Component, Input} from 'angular2/core'; @Component({ selector: 'cmp', template: ` <h1> Hello {{name}}</h1> `, inputs:['name'] }) export class Cmp { name: string; }
