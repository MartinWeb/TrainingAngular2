Main Concepts About Templates - What about 2 way data binding?

  • In ng1, 2 way data binding is an usefull feature which allows to bind data from an input.
  • What does it do: update the model from the input, and then update the view from the model, everything at the same time
  • The following seems to do what we need: Name: <input type="text" [value]="name" (input)="name= $event.target.name" >
  • Seems to be fine, but a shortcut syntax has been defined: [(ngModel)]="field"
  • Of if you don't like the [(ngModel)] syntax, use bindon-ngModel instead
import {Component} from '@angular/core'; import {bootstrap} from '@angular/platform-browser-dynamic'; import {SubCmp} from './sub-cmp'; @Component({ selector: 'app', template: '<sub-cmp name="john"></sub-cmp>', directives: [SubCmp] }) export class App{ name: string= 'john'; } import {Component, Input} from '@angular/core'; import {SubCmp} from './sub-cmp'; @Component({ selector: 'sub-cmp', template: ` <h1> Name: {{name}}</h1> <input type="text" [(ngModel)]="name" > ` }) export class SubCmp { @Input() name: string; }
