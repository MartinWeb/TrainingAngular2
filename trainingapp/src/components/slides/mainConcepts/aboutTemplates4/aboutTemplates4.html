Main Concepts About Templates - Property Binding

Your app is a tree of Web Components, and we need a way to parameterize our components. This is done by using the Property Binding mechanism.

The properties between brackets used on a component in the html are bind to the properties of the component object.

Let's try to set a name property to our custom SubCmp component: What do you see?

import {Component} from 'angular2/core'; import {bootstrap} from 'angular2/platform/browser'; import {SubCmp} from './sub-cmp'; @Component({ selector: 'my-app', template: '<sub-cmp name="john"></sub-cmp>', directives: [SubCmp] }) export class AppComponent {} bootstrap(AppComponent), []); import {Component, Input} from 'angular2/core'; import {SubCmp} from './sub-cmp'; @Component({ selector: 'sub-cmp', template: 'Hello {{name}}' }) export class SubCmp { @Input() name: string; }
