All previous notions can be used in your apps, but generally you will not need to use the injector object. The declaration of providers will be deported:

  • To the application bootstrap
  • To the component declaration

Bootstrap is the function to use, as its name says, to bootstrap an Angular2 application. Below is an example of its more basic usage

import {Component} from '@angular/core'; import {bootstrap} from '@angular/platform-browser-dynamic'; @Component({ selector: 'main-cmp', template: 'Hello World' }) class MainCmp{ constructor(){} } bootstrap(App);
