Main Concepts About Templates - Declare your components

  • You can use core Components/Directives into your templates without extra work
  • But you need to declare them if you want to use custom Web Components
  • To do that, there is a directives attribute on @Components

As an example, we are using: the ngFor

  • ngFor without any declaration
  • SubCmp which is declared in directives
import {Component} from 'angular2/core'; import {bootstrap} from 'angular2/platform/browser'; import {SubCmp} from './sub-cmp'; @Component({ selector: 'my-app', template: '<sub-cmp name="john"></sub-cmp>', directives: [SubCmp] }) export class AppComponent {} @Component({ selector: 'sub-cmp', template: `hello` }) export class SubCmp {} bootstrap(AppComponent), []);
