Main Concepts About Templates - Introduction

Compared to Angular1, the Angular2 templates syntax seems a little bit more complex. It looks like someone want to decorate the HTML with characters we are not used to see there...

As an example the following syntax is correct

<h1> {{ title }} </h1> <p>User List: </p> <ul> <li *ngFor="#user of users; #index= index;" (click)="goToDetails()" > <user-cmp [id]="user.id"></user-cmp> </li> </ul> <input type="text" [(ngModel)]="filterCriteria">

And yes, this is valid ...
