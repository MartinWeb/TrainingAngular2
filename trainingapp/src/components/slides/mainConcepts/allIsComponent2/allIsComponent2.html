Angular2 Services are just ES6 classes, and nothing more! Below is a naive implementation of a service to manage logs. Nothing more is needed to declare a service.

enum LogeLevel { INFO, WARN, ERROR } class LoggerService { private minLevel: LogLevel; constructor(minLevel?: LogLevel){ if (minLevel === undefined){ this.minLevel= LogLevel.WARN; } } info(msg: String): void{ if (this.minLevel >= LogLevel.INFO){ console.log('[INFO]' + msg); } } warn(msg: String): void{ if (this.minLevel >= LogLevel.WARN){ console.log('[WARN]' + msg); } } error(msg: String): void{ console.log('[WARN]' + msg); } }
