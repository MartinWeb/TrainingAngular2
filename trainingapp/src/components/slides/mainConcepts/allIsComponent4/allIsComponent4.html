Main Concepts Components Are Everywhere - Pipes

Pipes are the equivalent to filters in Angular1, they allow to apply a treatment to an entry value, eventually with parameters, and return a value.

@Pipe({ name:'truncate' }) export class Truncate{ private defaultIndex= 4; transform(val: string, index?: number){ var idxToUse= (index === undefined? this.defaultIndex: index); return val.substr(0, idxToUse); } }
