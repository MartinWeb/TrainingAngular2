Main Concepts Forms - Controls and ControlGroups

  • Based on the previous example, we are able to play with css classes, and for example, set a red border when the ng-invalid css class is on the input
  • But behind the ngControl directive, there is an Control instance, how can we access it to work with its properties dynamically
  • To acceed controls, the easier way is to get access to its parent ControlGroup instance. Any Control is child of ControlGroup
  • In our previous example, this worked naturally because Form is in fact a selector for the ngForm directive, which creates its own ControlGroup
  • To get the parent ControlGroup, the easier way is to create our own
    • ngFormModel to access the ngForm component and so its Controlgroup
import {Component} from 'angular2/core'; import {AbstractControl} from 'angular2/common'; import {bootstrap} from 'angular2/platform/browser'; import {MyFormCmp} from 'app/components/form-cmp'; @Component({ selector: 'my-app', template: '<my-form-cmp></my-form-cmp>', directives: [MyFormCmp] }) export class AppComponent {} bootstrap(AppComponent), []); import {Component} from 'angular2/core'; enum TodoCategory{PRIVATE, PERSO}; class TodoModel{ constructor(public category: TodoCategory, public details: string){} } @Component({ selector:'my-form-cmp', templateUrl:'app/components/form-cmp.html' }) export class MyFormCmp{ categories: string[]= Object.keys(TodoCategory).filter(v => isNaN(parseInt(v, 10))); model: TodoModel= new TodoModel(); isSubmit: false; details: AbstractControl; onSubmit(){ this.isSubmit= true; } } <h1>Add an element to your todo list</h1> <form (submit)="onSubmit()"> <p> <label for="category"> Category </label> <select [(ngModel)]="model.category" id="category" required> <option *ngFor="#category of categories" [value]="category">{{category}}</option> </select> </p> <p> <label for="todoDetails">Content</label> <input required ngControl="details" #detailsInput [(ngModel)]="model.details" id="todoDetails" type="text"> </p> <button type="submit">Submit</button> </form> <p *ngIf="isSubmit"> Form submited </p> <h2>Result</h2> <p *ngIf="!isSubmit"> {{model.category}} : {{model.details}} </p> <p> Classes: {{detailsInput.className}} </p>
