Main Concepts Dependency Injection - More Details - Bootstrap

But you can also provide services during the bootstrap of your application. These services will then be available everywhere in the app ( or a new instance of them if you redefine a new provider through inheritance )

import {Component} from 'angular2/core'; import {bootstrap} from 'angular2/platform/browser'; import {Person} from 'services/Person'; @Component({ selector: 'main-cmp', template: 'Hello World' }) class MainCmp{ constructor(public person: Person){} } bootstrap(App, [Person]);
