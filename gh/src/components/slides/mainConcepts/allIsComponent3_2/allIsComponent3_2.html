Main Concepts Classes Are Everywhere - Directives / Components

  • Directives Are just ES6 classes, as services
  • Some metadata is used to declare them and to add them some behavior

Below is a sample of Directives and Component

/** * A directive to reference content of an element * in order to reference them later * * ### Example * <div eltMarker [markerKey]="personContent"> * some content * </div> */ @Directive({ selector: '[etlMarker]' inputs: ['markerKey'] }) class ElementMarker{ constructor(private viewContainer: ViewContainerRef, private markerRefs: MarkerRefs){} // If you need more control on values set set markerKey(val: string) { this.markerRefs.add(val, this.viewContainer); console.log(val); } } @Component({ selector: 'hello-world', template:'<span>Hello name </span>', inputs: ['name'] }) class Redisplay{ public name: string='world'; constructor(){} }
