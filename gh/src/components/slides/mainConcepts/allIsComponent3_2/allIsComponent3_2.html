Main Concepts Classes Are Everywhere - Directives / Components

  • Directives Are just ES6 classes, as services
  • Some metadata is used to declare them and to add them some behavior

Below is a sample of Directives and Component

/** * A directive to listen double/triple/... clicks * * ### Example * <div [nth-clicks]="2" #nthClicked *ngIf="nthClicked.done" > * some content * </div> */ @Directive({ selector: '[nth-clicks]' inputs: ['markerKey'], host: { '(click)': 'clickedOnce()', }, exportAs:'nthClicked' }) export class NthClicks{ done: boolean= false; clickedOnce(){ //Check time and update done } } @Component({ selector: 'hello-world', template:'<span>Hello name </span>', inputs: ['name'] }) class Redisplay{ public name: string='world'; constructor(){} }
