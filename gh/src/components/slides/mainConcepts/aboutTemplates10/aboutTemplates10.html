Main Concepts About Templates - Declare your components

  • You can use core Components/Directives into your templates without extra work
  • But you need to declare them if you want to use custom Web Components
  • To do that, there is a directives attribute on @Components
  • In the example below we are using ngFor (not declared) and SubCmp (declared)
import {Component} from '@angular/core'; import {bootstrap} from 'angular2/platform/browser'; @Component({ selector: 'sub-cmp', template: `hello {{name}}`, inputs: ['name'] }) class SubCmp {} @Component({ selector: 'my-app', template: ` <ul> <li *ngFor="#name of ['john', 'donald']"> <sub-cmp [name]="name"></sub-cmp> </li> </ul> `, directives: [SubCmp] }) export class AppComponent {} bootstrap(AppComponent), []);
