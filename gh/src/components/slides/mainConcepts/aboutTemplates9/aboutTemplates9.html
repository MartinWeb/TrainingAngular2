Main Concepts About Templates - What about 2 way data binding?

For Angular1 users, 2 way data binding is a usefull feature which allows to bind data from an input. The idea is to update the model value from the input, and then update the view from the model, everything at the same time.

The following seems to do what we need:

Name: <input type="text" [value]="name" (input)="name= $event.target.name" >

That's not so difficult, but a shortcut syntax has been defined with the ngModel directive

import {Component} from '@angular/core'; import {bootstrap} from 'angular2/platform/browser'; import {SubCmp} from './sub-cmp'; @Component({ selector: 'my-app', template: '<sub-cmp name="john"></sub-cmp>', directives: [SubCmp] }) export class AppComponent { name: string= 'john'; } bootstrap(AppComponent), []); import {Component, Input} from '@angular/core'; import {SubCmp} from './sub-cmp'; @Component({ selector: 'sub-cmp', template: ` <h1> Name: {{name}}</h1> <input type="text" [(ngModel)]="name" > ` }) export class SubCmp { @Input() name: string; }
