Main Concepts About Templates - Property Binding - the basics

  • An ng2 app is a tree of Web Components
  • To be portable, a component has to be prameterizable
  • This is done by using the Property Binding mechanism

Let's try to set a name property to our custom SubCmp component:

import {Component} from '@angular/core'; import {bootstrap} from '@angular/platform-browser-dynamic'; import {SubCmp} from './sub-cmp'; @Component({ selector: 'app', template: '<sub-cmp name="john"></sub-cmp>', directives: [SubCmp] }) export class App {} import {Component, Input} from '@angular/core'; @Component({ selector: 'sub-cmp', template: 'Hello {{name}}' }) export class SubCmp { @Input() name: string; }
