Main Concepts Classes Are Everywhere - About Services - Existing ones

  • Services can be used everywhere in your app
    • In other services
    • In components/directives
    • In pipes
  • Dependency Injection details come in a few slides, for now consider than to use a service, you have to:
    • Declare it at the application bootstrap to make it available in the app import {MyAppComponent} from './my-cmp.ts'; import {MyService} from './my-service.ts'; bootstrap(MyAppComponent, [MyService]);
    • Declare it into the constructor of your directives/services to retrieve it at some point export class MyAppComponent{ constructor(private service: MyService){} //... }
