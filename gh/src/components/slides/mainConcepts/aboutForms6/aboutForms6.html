Main Concepts Forms - Custom Validators

  • In previous samples, we used the required validator with Control / ControlGroup and FormBuilder tools to add some behavior when something goes wrong! (no data into the input)
  • The native validators provided by Angular2 are:
    • required, minlength, maxlength
    • pattern: to be sure that a form Control match a specific pattern (same syntax than js regex)
  • Finally, we need a way to create our own Validator
