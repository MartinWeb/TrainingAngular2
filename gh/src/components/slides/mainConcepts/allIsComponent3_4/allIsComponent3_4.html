Main Concepts Classes Are Everywhere - Directives / Components - Templates Overview

  • This is a first short overview of templates (let's see them into details soon)
  • We have seen that templates have to be declared in our @Component annotation
  • And that they can use public components methods or attributes
  • The example below is a component with its template, which also contain a reference to an other directive
@Component({ selector: 'hello-world', templateUrl:'hello-world.html', inputs: ['name'] directives:[NthClicks] }) class Redisplay{ private defaultName: string= 'Bob'; public name: string='Bob'; //default value constructor(){} resetName(){ this.name= defaultName; } } <h1>Hello World Template Content</h1> <p> I'm used to say hello to somebody, so hello {{name}}! </p> <button (click)="resetName()">Click Me To reset the name</button> <div class="redBox" [nth-clicks]="3" #nthClicked> Try to triple click me!!! <span *ngIf="nthClicked.done"> Hey, you did it </span> </div>
