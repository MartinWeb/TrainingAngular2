Main Concepts Dependency Injection - Introduction

We have seen the basic types of elements available in Angular2, and missed one of the important feature provided, the Dependency Injection.

Basically, Dependecy Injection (DI), also called Inversion Of Control (IOC) is not specific to Angular2, it's a common pattern used by many frameworks and whose the goal is to minimize the modifications of code when an implementation changes and to let the framework manage instantiations for you.
