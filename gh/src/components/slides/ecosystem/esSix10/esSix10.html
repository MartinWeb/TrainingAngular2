Ecosystem ES6 Introduction - Templates literals

If you come from the java world (and in fact, from most of other languages), you are probably used to format Strings with specific methods. In javascript, most of time, this was done by just concatenating the strings you need. From ES6, you will have a new machanism for that (which hunappily remains less powerfull than it could)

class Person{ constructor(firstName, lastName){ this.firstName= firstName; this.lastName= lastName; } descriptionOldVersion= () => '[ ' + this.firstName + ' , ' + this.lastName + ' ]'; //backticks instead of quotes, and possibly multi lines descriptionNewVersion= () => `[ ${firstName} ${lastName} ]`; }
