Preamble Angular2 VS Angular1, what remains? How to port from 1 to 2?

Some concepts remain, even if their implementation is fully different

  • Template expressions, {{expression}}, one way data binding
  • 2 way data binding can be simulated with help of events
  • services
  • The notions of directive, even if its name and usage changes
  • Automatic change detection is still available, but is managed differently by the framework (events)
  • filters are now named pipes
